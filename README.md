# Historium Universe

Repositorio maestro de contenido para Historium.

Aquí vive la biblia editorial, campañas, personajes, ciudades, batallas, reliquias, conceptos históricos, prompts visuales y plantillas de producción.

## Estructura

- `00_Biblia_Editorial/` — reglas narrativas, tono, sistema de XP, rarezas y estilo.
- `01_Game_Design/` — filosofía de juego, progresión, recompensas, economía y features futuras.
- `02_Campaigns/` — campañas completas divididas por civilización, periodo o personaje.
- `03_Knowledge/` — base de conocimiento histórico reutilizable.
  - `Characters/`
  - `Cities/`
  - `Battles/`
  - `Concepts/`
  - `Civilizations/`
  - `Empires/`
  - `Religions/`
  - `Documents/`
  - `Monuments/`
  - `Artifacts/`
  - `Technologies/`
  - `Biographies/`
- `04_Templates/` — formatos oficiales para nodos, campañas y fichas.
- `05_Prompts/` — prompts visuales, audio, video y assets.
- `06_Importer/` — proceso de producción e importación.
