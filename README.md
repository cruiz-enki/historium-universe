# Historium Universe

Repositorio maestro de contenido para Historium.

Este repositorio contiene el Canon Oficial del Universo Historium: biblia editorial, game design, especificaciones, conocimiento, campañas, prompts, importer, assets y plantillas.

Historium no produce lecciones.

Historium construye un Museo Vivo del conocimiento histórico.

---

# Qué es Historium

Historium es una plataforma educativa tipo videojuego narrativo/RPG para aprender Historia.

Combina el ritmo de una serie premium, la progresión de un RPG, la claridad de una app moderna y la profundidad de un museo vivo.

No es Wikipedia.

No es un curso genérico.

No es una enciclopedia tradicional.

Es un universo persistente donde cada campaña cuenta una historia y cada entidad alimenta el Museo del jugador.

---

# Separación clave

- GitHub contiene el Canon Oficial.
- Supabase contiene el progreso del usuario.
- Las campañas cuentan historias.
- Las entidades almacenan conocimiento.
- Los nodos actualizan el Museo.

---

# Estructura

- `00_Biblia_Editorial/` — identidad, manifiesto, tono, patrones narrativos y guías editoriales.
- `01_Game_Design/` — filosofía de juego, progresión, recompensas y sistema de conocimiento.
- `02_Specifications/` — reglas de producción para campañas, nodos y entidades.
- `03_Knowledge/` — base del Grafo Histórico y galerías del Museo.
- `04_Campaigns/` — campañas narrativas oficiales.
- `05_Prompts/` — prompts de imagen, audio, video, mapas, personajes y artefactos.
- `06_Importer/` — reglas para convertir Markdown canónico en datos importables.
- `07_Assets/` — materiales fuente y referencias.
- `08_Templates/` — plantillas oficiales de producción.
