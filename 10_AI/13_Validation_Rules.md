# HISTORIUM — AI VALIDATION RULES

**Versión:** 1.0  
**Estado:** Oficial

---

# Checklist automático

- [ ] ID válido.
- [ ] Frontmatter completo.
- [ ] Entidades existentes verificadas.
- [ ] Relaciones válidas.
- [ ] Sin duplicados.
- [ ] Markdown importable.
- [ ] Tono Historium.
- [ ] Rigor histórico.
- [ ] Museo actualizado.
- [ ] Fuentes declaradas.
- [ ] Assets conectados.

---

# Errores bloqueantes

- ID inventado.
- Fuente inventada.
- Relación libre.
- Entidad duplicada.
- Nodo sin Museo.
- Quiz memorístico vacío.
- Imagen con texto incrustado.

---

# Regla Suprema

La validación existe para proteger al jugador de contenido débil.
