# HISTORIUM — ASSETS

**Estado:** Oficial

---

# Propósito

Esta carpeta organiza materiales visuales, sonoros y fuentes auxiliares del universo Historium.

Los assets deben estar conectados a campañas, entidades, nodos o prompts.

---

# Principios

- Cada asset debe tener función narrativa o museística.
- Cada asset debe poder rastrearse a una entidad o campaña.
- Cada asset debe respetar el tono visual premium de Historium.
- Cada asset debe evitar estereotipos históricos.

---

# Regla Suprema

Un asset que no aumenta comprensión no entra al universo.
